# Application de recommandation de films
Ce projet est réalisé dans le cadre de la formation DATA ANALYST de la WILD CODE SCHOOL.

Les technologies et notions utilisées dans ce projet :
- technos : Python, Pandas, Seaborn, Streamlit, Git/Github, HTML/CSS
- notions data : Data-visualisation, Machine Learning


# Installation
1. Cloner le projet en local : `git clone https://github.com/kenjivictor/projet_recommandation_films.git`
2. Se déplacer dans le dossier créé : `cd projet_recommandation_films`
3. Installation des packages avec uv : `uv sync`
4. Activation de l'environnement : `source .venv/Scripts/activate`


# Fonctionnalités principales
- Authentification sur l'application
- Films à l'affiche
- Recherche de films par critères (Producteurs, Acteurs, Genre, etc...)
- Dataviz de la base de films
